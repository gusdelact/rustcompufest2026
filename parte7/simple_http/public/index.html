¿Qué es Simple HTTPD?

Es un servidor HTTP minimalista construido desde cero en Rust, diseñado para servir archivos estáticos de forma rápida y segura. Aprovecha las garantías de seguridad de memoria de Rust y su modelo de concurrencia sin data races para ofrecer un servidor confiable y eficiente.

⚡

Rápido

Rendimiento nativo sin garbage collector

🔒

Seguro

Memory safety garantizada por el compilador

🪶

Ligero

Mínimo uso de recursos del sistema

🔧

Simple

Fácil de configurar y ejecutar

Inicio rápido

// Compilar y ejecutar
$ cargo build --release
$ ./target/release/simple-httpd --port 8080 --root ./public

// O directamente con cargo
$ cargo run -- --port 8080

Ejemplo de código

use std::net::TcpListener;
use std::io::prelude::*;

fn main() {
    let listener = TcpListener::bind("127.0.0.1:8080").unwrap();
    println!("Servidor escuchando en puerto 8080");

    for stream in listener.incoming() {
        handle_connection(stream.unwrap());
    }
}
